Datei-Verbindung

Auto-Save direkt in deine Drive-Datei — die Synchronisation macht Drive Desktop.

Status

—

So funktioniert's

  1. Google Drive für Desktop installieren (google.com/drive/download) — Drive-Ordner erscheint dann als normaler Finder-Ordner.
  2. Diese HTML-Datei und die JSON-Datei legst du gemeinsam in einen Ordner deines Drives.
  3. HTML in Chrome/Edge per Doppelklick öffnen, dann unten auf „Datei einrichten" → JSON auswählen → „Zulassen". Fertig.
  4. Auto-Save läuft. Drive Desktop synct die geänderte JSON in die Cloud, die zweite Person sieht's beim nächsten ⌘R.

⚠️ Funktioniert nur in Chromium-Browsern (Chrome, Edge, Brave) — Safari und Firefox unterstützen die File System Access API nicht.
